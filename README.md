# AdvancedCompany Unofficial

Unofficial compatibility patch/fork of AdvancedCompany for Lethal Company v80.

I am not the original author of AdvancedCompany.

Original mod: AdvancedCompany  
Original author/project: FluffyFishGames / PotatoePet  
Original repository: https://github.com/FluffyFishGames/AdvancedCompany  
Original license: CC0-1.0

## What this fork does

This fork attempts to make the legacy AdvancedCompany mod work on Lethal Company v80.

## Current status

This is an early compatibility test build.

Known issues:

- Voice filter update is currently disabled to prevent `SoundManager.SetPlayerVoiceFilters` errors.
- Voice chat may work, but distance/filter effects may be inaccurate.
- Compatibility with other mods is not guaranteed.
- Bugs are expected.
